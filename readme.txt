This plugin helps you take notes to write your post!

== Description ==

This plugin puts a notes metabox on the side of your post edit screen to help you take notes and more easily write your post!

== Installation ==

Add this plugin by uploading the zip using the "Add Plugin" feature built into Wordpress.  Otherwise manaully unzip the folder
and upload the entire directory to your blog's plugins folder ( /wp-content/plugins/ ).  Next you have to activate the plugin.

Alternatively you could just go to "Add Plugin" and search for the Wordpress.org Plugins and directly install it from there.

== Frequently Asked Questions ==

= How do I use this plugin? =

Whenever you are editing a post or a page you will see a metabox in the upper right-hand corner where you can toggle displaying or not displaying the post list.

You can set some other options there as well, such as the category for posts you want to display, how many posts to display, whether or not to show an excerpt, and how many words long the excerpt should be.
